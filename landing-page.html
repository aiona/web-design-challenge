Weather Data Analysis


Project Overview


The purpose of this project was to gather weather data from over 500 cities around the world using Open Weather Map's API. The results were then stored in a Pandas dataframe. A central question the data tried to answer was, "What happens to weather as you approach the Equator?" The answer seems obvious, that the weather gets warmer. But this answer is much less interesting, and less satisfying without data to back it up, and to provide greater context.

Once the data was collected, each of four weather factors were plotted to show how temperature, wind speed, humidity and cloud coverage change as latitude increases. Of these, the relationship is clearest between the latitude and the maximum temperature of a given city. As the analysis shows, while there is some correlation between latitude and humidity, wind speed and cloud coverage, it seems that to the extent latitude does impact these other factors, there are other variables with greater influence.
